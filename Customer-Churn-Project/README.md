# Customer Churn Prediction and Retention Strategy Optimization

## Project Overview
This project predicts telecom customer churn using Machine Learning and provides retention strategies through an interactive Power BI dashboard.

## Technologies Used
- Python
- Google Colab
- Random Forest
- Gradient Boosting
- Power BI

## Project Workflow
Dataset → Data Preprocessing → Churn Prediction → Risk Segmentation → Retention Strategy → Business Simulation → Dashboard

## Features
- Customer churn prediction
- Risk segmentation
- Retention strategy recommendation
- Business simulation
- Interactive dashboard

## Machine Learning Models
- Random Forest
- Gradient Boosting

## Dashboard Pages
1. Churn Overview
2. Customer Churn Analysis
3. Risk Segmentation
4. Retention Strategy
5. Business Simulation

## Dataset
IBM Telco Customer Churn Dataset
